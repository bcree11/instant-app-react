# Page

### Constructor:

#### new **Page(_properties?_)**

##### Property Overview:

| Name            | Type    | Summary                           |
| --------------- | ------- | --------------------------------- |
| backgroundImage | Image   | Image or URL of background image. |
| title           | String  | Title text.                       |
| subtitle        | String  | Subtitle text.                    |
| scrollable      | Boolean | Toggle option to scroll.          |
